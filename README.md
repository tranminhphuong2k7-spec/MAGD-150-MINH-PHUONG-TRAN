# MAGD-150-MINH-PHUONG-TRAN
### Lab 10: p5.anaglyph Proof-of-Concept
A simple 3D anaglyph scene built with p5.js and the p5.anaglyph library by Jenna deBoisblanc. View with red/cyan glasses for the full 3D effect.
The scene features a rotating box, an orbiting sphere, and a torus, all rendered using the Dubois anaglyph algorithm. Built as a lab proof-of-concept exploring stereoscopic 3D in the browser.

### Lab8: Holiday Greeting Card
A summer-themed interactive greeting card built with p5.js. A beach scene serves as the background, and a shark follows your cursor around the canvas. The shark scales in size based on vertical mouse position — move toward the top and it grows larger, move toward the bottom and it shrinks, simulating depth in the water.


### Lab7: Array & Random Value Explorer#
A p5.js sketch that demonstrates core JavaScript array methods and p5's random() function. Simulates a simple inventory/vending machine system using two arrays — item slots and prices — and displays randomly selected values on the canvas.
